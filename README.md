# 🖊️ Artools Precision Pen - AI Landing Page

Este projeto é uma landing page imersiva e de alta conversão para uma caneta tecnológica fictícia chamada "Artools Precision Pen". 

O grande diferencial deste repositório é o seu método de desenvolvimento: **todo o código, layout, imagens e vídeos foram gerados utilizando exclusivamente ferramentas e modelos de Inteligência Artificial**. 

O projeto foi desenvolvido como parte prática da **Formação de AI Designer da Asimov Academy**, com o objetivo de explorar os limites e o fluxo de trabalho das IAs generativas na criação de interfaces web completas e profissionais.

## 🤖 IAs e Ferramentas Utilizadas

A construção desta página foi inteiramente guiada por engenharia de prompts, utilizando o seguinte ecossistema de ferramentas:

* **Antigravity:** Responsável pela geração de toda a estrutura de código (HTML), estilização (Tailwind CSS) e pela complexa lógica de interações e animações em JavaScript.
* **Whisk:** Utilizado para a ideação e geração de todas as imagens e assets visuais estáticos (mockups) perfeitamente integrados ao layout.
* **Flow:** Ferramenta utilizada para a criação e renderização dos vídeos de alta fidelidade usados nos backgrounds e nas demonstrações de produto da página.
* **Bibliotecas de Front-end:** As IAs foram instruídas a implementar **Tailwind CSS** (para estilização utilitária e Dark Mode), **GSAP + ScrollTrigger** (para animações robustas baseadas em scroll) e **Lenis** (para um smooth scroll impecável).
